## Project SiEdukasi (SiEducation) 
=============================================================================================================<br>
<p>Contains educational projects targeted at everyone from elementary school to college for those who want to learn from the experiences of others.</p>
<p>Use tools to build in website SiEducation this:</p>
<ui>
  <li>Laravel</li>
  <li>Tailwind</li> 
  <li>HTML</li>
</ui>
Then I used a database called mysql and to enable mysql and apache I used the laragon tool.
